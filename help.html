使用指南

  1. 打开 首页，进入编辑画布。
  2. 左侧工具栏提供节点、连线、文本等基本元素，点击后在画布任意位置点击即可创建。
  3. 选中元素后，可在右侧属性面板修改颜色、文字、大小等属性。
  4. 使用鼠标拖拽或键盘方向键微调位置，滚轮缩放画布。
  5. 完成后可在 分享与导出 页面导出 PNG、JSON 或生成分享链接。

快捷键说明

  • Ctrl + Z – 撤销
  • Ctrl + Y – 重做
  • Del – 删除选中元素
  • Ctrl + C / Ctrl + V – 复制 / 粘贴
  • Ctrl + G – 组合选中元素
  • Ctrl + U – 解除组合
  • ← ↑ → ↓ – 微调选中元素位置（每次 1px）
  • Space + 拖动 – 暂时手掌模式，平移画布

常见问题

  • 如何保存我的流程图？
    在 分享与导出 页面点击“导出 JSON”，保存到本地，之后可在首页通过“导入 JSON”恢复。
  • 画布卡顿怎么办？
    请关闭不必要的浏览器标签页，或在属性面板中关闭节点阴影效果以提升渲染性能。
  • 想自定义节点样式？
    在属性面板中点击“高级样式”，可以手动编辑 CSS 代码，实时预览效果。
